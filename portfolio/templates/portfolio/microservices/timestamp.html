Timestamp Microservice

How to use:

  1. You can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016).
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null.

Examples:

  1. https://wawraf.pythonanywhere.microservices/December%2015,%202015
  2. https://wawraf.pythonanywhere.microservices/1450137600

Output:

  1. { "unix": 1450137600, "natural": "December 15, 2015" }
